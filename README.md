# 📸 Image Feed – Portfolio Front-end (Next.js)

This is a simple front-end project built with **Next.js**, simulating an image feed similar to social media platforms.

## ✨ Features

- 🖼️ Image feed layout built with reusable components
- ⚡ Fast rendering with Next.js
- 💅 Styled with Tailwind CSS


## 🛠 Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## 🔐 Environment Variables

Create a `.env.local` file based on the following template:

```env
NEXT_PUBLIC_API_URL=http://localhost:8080
NEXT_PUBLIC_WEB_URL=
NEXT_PUBLIC_IMAGE_URL=http://localhost:8080/static
```

## 🚀 Getting Started

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/ma-molinari/m-feed-web.git

# Enter the project directory
cd m-feed-web

# Install dependencies
npm install

# Start the dev server
npm run dev
